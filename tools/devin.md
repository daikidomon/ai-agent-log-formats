# Devin

## 基本情報

| 項目 | 内容 |
|------|------|
| 種別 | 自律AIエンジニア（クラウド） |
| ログ形式 | — |
| ローカルログ | なし |

## 保存場所

ローカルにログは保存されない。すべての会話・作業履歴は Cognition Labs のクラウドサービス上に保存。

## データ取得

- Web UI 上でセッション履歴を閲覧可能
- API経由でのエクスポートは公式ドキュメントを参照
- Slack 連携の場合は、Slack 側にメッセージ履歴が残る

## 注意事項

- 完全にクラウドベースのサービス
- ローカルからのログ収集は不可
- 組織の管理者がAPI経由で履歴を取得できる可能性がある
