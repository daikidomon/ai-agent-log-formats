# v0 (Vercel)

## 基本情報

| 項目 | 内容 |
|------|------|
| 種別 | ブラウザベースUI生成ツール |
| ログ形式 | — |
| ローカルログ | なし |

## 保存場所

ローカルにログは保存されない。Vercel のクラウドサービス上で動作。

## データ取得

- Web UI 上で生成履歴を確認可能
- 生成されたコードはコピー・エクスポート可能
- チャット履歴のエクスポート機能は限定的

## 注意事項

- 完全にブラウザ/クラウドベースのサービス
- ローカルからのログ収集は不可
- 主にUI/コンポーネント生成に特化
