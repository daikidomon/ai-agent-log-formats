# Windsurf (Cascade)

## Basic Information

| Item | Details |
|------|---------|
| Type | AI IDE |
| Log Format | Text (auto-summarized memory) |
| Local Logs | Yes (in memory format) |

## Storage Location

| OS | Path |
|----|------|
| Windows | `%USERPROFILE%\.codeium\windsurf\memories\` |
| macOS | `~/.codeium/windsurf/memories/` |
| Linux | `~/.codeium/windsurf/memories/` |

Backup (when using cascade-backup-utils): `~/.cascade_backups/`

## File Structure

```
~/.codeium/windsurf/
└── memories/
    └── {workspace-id}/
        └── (memory files)
```

## Data Format

Text files. Summaries and memories automatically generated by Cascade from conversations.

These are not the original prompts themselves, but summaries of information that Cascade deemed important.

## Retrieval Method

```python
from pathlib import Path

memories_dir = Path.home() / ".codeium" / "windsurf" / "memories"
for mem_file in sorted(memories_dir.rglob("*"), key=lambda p: p.stat().st_mtime, reverse=True):
    if not mem_file.is_file():
        continue
    text = mem_file.read_text(encoding="utf-8").strip()
    if text:
        print(f"[{mem_file.parent.name}] {text[:200]}")
```

## Notes

- Conversation logs themselves may not be saved locally
- `memories/` is separated by workspace
- Memory contents are summaries and differ from the original prompts
- Use file modification time as a substitute for timestamps
- If `~/.cascade_backups/` exists, it can also be referenced
