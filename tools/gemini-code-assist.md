| 項目 | 内容 |
|------|------|

## 保存場所（推定）

### VS Code拡張

| OS | パス |
|----|------|
| Windows | `%APPDATA%\Code\User\globalStorage\google.geminiaicodeassist\` |
| macOS | `~/Library/Application Support/Code/User/globalStorage/google.geminiaicodeassist/` |
| Linux | `~/.config/Code/User/globalStorage/google.geminiaicodeassist/` |

※ 拡張機能IDは推定。実際のIDは VS Code マーケットプレイスで確認が必要。

## データ形式

不明。Google Cloud との連携が深いため、会話履歴の大部分はクラウド側に保存される可能性が高い。
