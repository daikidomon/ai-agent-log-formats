# Bolt (StackBlitz)

## Basic Information

| Item | Details |
|------|---------|
| Type | Browser-based AI development environment |
| Log Format | — |
| Local Logs | None |

## Storage Location

No logs are stored locally. Everything runs in the browser, and project data is stored on StackBlitz's cloud.

## Data Retrieval

- Project history can be viewed on the Web UI
- Generated code can be downloaded and exported, but prompt history export functionality is limited

## Notes

- Entirely browser/cloud-based service
- Log collection from local machines is not possible
- When self-hosting the OSS version (bolt.diy), logs may be stored in a local database
