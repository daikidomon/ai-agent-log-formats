# Bolt (StackBlitz)

## 基本情報

| 項目 | 内容 |
|------|------|
| 種別 | ブラウザベースAI開発環境 |
| ログ形式 | — |
| ローカルログ | なし |

## 保存場所

ローカルにログは保存されない。ブラウザ上で完結し、プロジェクトデータは StackBlitz のクラウドに保存。

## データ取得

- Web UI 上でプロジェクト履歴を確認可能
- 生成されたコードはダウンロード・エクスポート可能だが、プロンプト履歴のエクスポート機能は限定的

## 注意事項

- 完全にブラウザ/クラウドベースのサービス
- ローカルからのログ収集は不可
- OSS版 (bolt.diy) を自己ホストする場合は、ローカルDBにログが保存される可能性がある
