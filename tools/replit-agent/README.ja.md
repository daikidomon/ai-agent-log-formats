# Replit Agent

## 基本情報

| 項目 | 内容 |
|------|------|
| 種別 | クラウドIDE内蔵エージェント |
| ログ形式 | — |
| ローカルログ | なし |

## 保存場所

ローカルにログは保存されない。会話履歴は Replit のクラウドサービス上に保存。

## データ取得

- Replit の Web UI 上でチャット履歴を閲覧可能
- Replit の `.replit` 設定ファイルや `replit.nix` はローカルにクローン可能だが、チャット履歴は含まれない

## 注意事項

- 完全にクラウドベースのサービス
- ローカルからのログ収集は不可
