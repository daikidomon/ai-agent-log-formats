# Replit Agent

## Basic Information

| Item | Details |
|------|---------|
| Type | Cloud IDE built-in agent |
| Log Format | — |
| Local Logs | None |

## Storage Location

No logs are stored locally. Conversation history is stored on Replit's cloud service.

## Data Retrieval

- Chat history can be viewed on the Replit Web UI
- Replit's `.replit` configuration file and `replit.nix` can be cloned locally, but chat history is not included

## Notes

- Entirely cloud-based service
- Log collection from local machines is not possible
