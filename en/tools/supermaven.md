# Supermaven

## Basic Information

| Item | Details |
|------|---------|
| Type | Code completion extension (VS Code / JetBrains / Neovim) |
| Log Format | — |
| Local Logs | Limited |
| Remarks | Primarily completion-based, so there are almost no conversation logs |

## Storage Location

As a VS Code extension:

| OS | Path |
|----|------|
| Windows | `%APPDATA%\Code\User\globalStorage\supermaven.supermaven\` |
| macOS | `~/Library/Application Support/Code/User/globalStorage/supermaven.supermaven/` |
| Linux | `~/.config/Code/User/globalStorage/supermaven.supermaven/` |

## Data Format

Supermaven's primary function is code completion, and it generally does not generate chat-style conversation logs.

## Notes

- A tool specialized in fast code completion
- As it does not have a chat feature, it is not well suited for prompt analysis
- Supermaven was acquired by Cursor in 2024, so it may be integrated in the future
